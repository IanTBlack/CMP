This folder contains recovered data from the CMP deployed at the commercial docks in Yaquina Bay.

Recovered data files are named: RXXXXX-YYYY-MM-DD
where, XXXXX is the sequence number of the recovery. DXXXXX files with equivalent numbers are the deployment specifications for the recovered data,
       YYYY is the year the data was recovered,
       MM is the month the data was recovered,
       DD is the day the data was recovered,

These files contain raw data from each instrument.

The file named Yaquina_Dock5_Compiled contains a running file of recovered data, edited to remove test data. This file will show gaps in between recovery operations.
